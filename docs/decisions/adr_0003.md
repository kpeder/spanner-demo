## 3. Code is PEP8 compliant

Date: 2026-03-25

### Status
Accepted

### Context
PEP8 is the official style guide for Python. Package code should be PEP8 compliant. Flake8 is the defacto tool for PEP8 standards checking.

### Decision
All Python code must be PEP8 compliant, and pass flake8 linting tests.

### Consequences
Flake8 linting checks will need to pass before any Pull Request is merged.
